Virhe:
   "A graphics object cannot be created from an image that has an indexed pixel format"
Syy:
   Kuvan lataaja vaatii 24-bittiset värit.

Poikkeus:
   NotImplementedException Content.Load -rivillä
Ympäristö:
   VS2012 WP8
Syy:
   WP8-versio ei osaa kääntää contenttia.
Ratkaisu:
   Käännä xnb:ksi XNA Content Compilerilla.
   https://xnacontentcompiler.codeplex.com/

Virhe:
   "Error CS2001: Source file `opentk/Source/GlobalAssemblyInfo.cs' could not be found (CS2001) (OpenTK)"
Ympäristö:
   Linux/MonoDevelop
Ratkaisu:
   OpenTK pitää kääntää ensin konsolin kautta
   xbuild OpenTK.sln /p:Configuration=Release
