"Projektimallit" MonoDevelopille (Linux)
-----------------------------------------

1. Ota kopio kansiosta jonka pohjalta haluat tehdä pelin
	PerusPeli = peli ilman fysiikkaa
	FysiikkaPeli_SimplePhysics = peli yksinkertaisella fysiikalla (jos haluat tehdä pongin tms.)
	FysiikkaPeli_Physics2d = peli täyden fysiikan kanssa
	Tasohyppelypeli = valmis tasohyppelypohja norsun kanssa

2. Nimeä projekti uudelleen
	Kansion nimi
	Avaa MonoDevelopilla
		Vaihda solutionin nimi ja projektin nimi solution-ikkunasta vasemmalta
		Ota projektista oikealla painikkeella Options
			Mene Output-välilehdelle ja vaihda Assembly name

3. Avaa Peli.cs ja ala koodata :)


Tiedostojen lisääminen contentiin
----------------------------------

1. Varmista että tiedostot jotka haluat lisätä ovat pelin content-kansiossa.

2. Valitse vasemmalta Solution-ikkunasta Content ja kontekstivalikosta (= oikealla hiirenpainikkeella)
	Add -> Add Files

3. Valitse lisättävät tiedostot ikkunasta ja hyväksy ok:lla.

4. Jos et näe Solution-ikkunan alla Properties-näkymää, valitse lisäämäsi tiedoston kohdalta kontekstivalikosta Properties

5. Valitse properties-näkymästä *jokaiselle lisäämällesi tiedostolle*
	Build action = Content
	Copy to output directory = Copy if newer
